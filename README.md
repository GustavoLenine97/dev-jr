# Roadmap Profissional de 60 Dias para Conquista da Primeira Vaga como Desenvolvedor

## Objetivo

Preparar um profissional formado em ADS, sem experiência profissional, para disputar vagas de Desenvolvedor Júnior utilizando práticas reais de mercado.

Este roadmap foi desenhado para simular a rotina de uma equipe moderna de desenvolvimento em 2026.

Carga horária:

- 2 horas por dia
- 60 dias
- Aproximadamente 120 horas de prática

---
